VF Web - Landing Page (Bootstrap)

Como usar local:
1) Abra o arquivo index.html no navegador.
2) Para testar como se estivesse hospedado, você pode usar um servidor simples.

Opção A (Python):
  python -m http.server 8080
  Abra: http://localhost:8080

Opção B (VS Code):
  Extensão "Live Server" -> Go Live

Personalização rápida:
- Cor principal: assets/css/style.css -> --vf-primary
- WhatsApp: procurar por "wa.me" no index.html
- Links (GitHub/LinkedIn/Instagram): ajustar no index.html

Observação:
Esta LP é estática. O formulário está em modo demo e precisa de um endpoint (PHP, Formspree etc.) para enviar de fato.
